How to Use GRAPS

GRAPS is an interactive graph visualization and algorithm exploration tool. Here’s a quick overview to get you started:

🎲 Random Graph Generation (RGG Section)

  • Only unweighted graphs are supported in this section.
  • Use the ⚙️ settings icon to configure detailed options for generating random graphs.

✏️ Graph Creation Mode

  • Create nodes and edges by clicking on the canvas.
  • Shift + Click on a node to delete it.
  • Shift + Click on an edge to delete it.

🎨 Color Settings & Controls

  • Customize edge colors and highlight specific edges for clarity.
  • Use the 🧹 Clear button to reset the graph.
  • Save your current graph to 💾 Local Storage for easy access later.

📂 Examples

  • Import quick pre-made example graphs to explore.
  • Manage your 📁 Local Saved Examples (up to 10), with options to delete individual saved graphs.

⚙️ Available Algorithms

  • Force-Directed Layout using Eades Spring Embedder
  • Paths using DFS-based methods
  • Dijkstra’s Shortest Path
  • Minimum Spanning Trees (Kruskal’s and Prim’s Algorithms)
  • Breadth-First Search (BFS)
  • Depth-First Search (DFS)
  • All Available Paths (DFS-Based)

🔄 Import / Export

  • Import and export graphs using JSON format.
  • Copy graph JSON data to clipboard for sharing or backup.

💡 Psst... don't forget to hover over the ❤️ heart of GRAPS — it has something to tell you every time!
